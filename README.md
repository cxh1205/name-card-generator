# 三角名牌生成器 · Tent Card Generator

在线生成可折叠三角形名牌的 Web 工具，输入名字列表即可一键打印。

## 效果

每张名牌是一张正方形纸张，横向一分为二：

- **上半部分**：名字倒置打印（折叠后成为背面视角的正向文字）
- **下半部分**：名字正向打印（正面视角）

沿中线折叠后形成三角形立牌，从正面和背面均可看到正向的名字。

## 功能

- 批量输入名字（换行 / 逗号 / 顿号分隔）
- 楷体排版，空格控制字间距
- 自动撑满模式 — 字号根据字数自动适配半张卡片
- 手动百分比模式 — 字号占半卡高度的 10%–95% 可调
- 自定义背景颜色 / 背景图片（自动居中裁剪为 2:1）
- 自定义字体颜色、粗细
- 1/2/3 列预览切换，卡片自适应右栏宽度
- 打印纸张边长独立可调，每张名牌独立分页
- 打印时自动隐藏控制面板和分割线

## 技术栈

Vue 3 + Vite · CSS Container Queries · Canvas 图片裁剪

## 使用

```bash
npm install
npm run dev
```

打开浏览器，左侧输入名字并调整样式，右侧实时预览。点击底部「打印」按钮或 `Ctrl+P` 打印，浏览器打印对话框中选择 **A4 纸张**，每张名牌自动独占一页。打印后沿正方形边缘裁剪，沿中线折叠即可。
